# social-auto-upload4pkg
本项目fork自[social-auto-upload](https://github.com/dreammis/social-auto-upload)，为了方便其它工程集成所以做了少量修改。 


# 💾Installation
``` shell
# 打包
python setup.py sdist
# 安装
pip install dist/social_auto_upload-1.0.0.tar.gz
```
